Aceite pagamentos em PIX e cartão com o Rappibank, gateway de pagamento do Rappi.

== Description ==

Pagamentos via Pix ou cartão de crédito com as melhores taxas do mercado.

O pagamento via Pix acontece de maneira instantânea dando notificação no momento do pagamento para o cliente que pagou.

Reembolsos podem ser realizados tanto pelo woocommerce quanto pela plataforma, que interage com o seu site e atualiza os pedidos.

Esse plugin utiliza os serviços do [Rappibank](https://rappibank.com.br) para processar os pagamentos. 

Para mais informações sobre quais dados e como utilizamos esses dados, consulte nossa [Política de privacidade](https://bank.rappi.com.br/termos-e-condicoes/artigo/politica-de-privacidade)

== Frequently Asked Questions ==

= Por quê vejo a mensagem "Não há métodos de pagamento disponíveis."? =

Isso pode ser porque a sua página de finalização de compra está configurada como blocos, e o nosso plugin ainda não aceita blocos. Para corrigir, vá em Páginas no menu, e clique em Editar a página de <strong>Finalização de compra</strong>, selecione o bloco de opções de pagamento e no menu à direita, clique em <strong>Mudar para finalização de compra clássico</strong>.

= Como faço para me cadastrar? = 

Para se cadastrar, acesse o site do [Rappibank, Brasil](https://rappibank.com.br), e clique no botão "Criar conta".
